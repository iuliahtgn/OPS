Linear / Lineares Optimierungsprogramm

Linear

Linearität ist die Eigenschaft eines Systems, auf die Veränderung eines Parameters stets mit einer dazu proportionalen Änderung eines anderen Parameters zu reagieren.

Ein System ist linear wenn:

  1. Wenn aus der Eingabe x im System die Ausgabe X resultiert, dann eine Eingabe von 2x soll in einer Ausgabe von 2x resultieren. TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT TEXT v
  2. Wenn aus der Eingabe x im System die Ausgabe X resultiert und aus der Eingabe y im System die Ausgabe Y resultiert, dann eine Eingabe von x+y soll in eine Ausgabe von X+Y resultieren. Das heißt, dass das System die 2 Eingaben unabhängig voneinander simultan abarbeitet und sie interagieren nicht miteinander im System.
